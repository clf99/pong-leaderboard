Pong Leaderboard

Shared leaderboard module for Cliffella Pong games.

Module URL

https://cdn.jsdelivr.net/gh/clf99/pong-leaderboard@main/leaderboard.js

What it does

  • Stores shared scores on Nostr relays instead of local-only browser storage
  • Lets AIPong publish volley scores
  • Lets classic Pong publish shared match results and standings

Example

import * as Leaderboard from 'https://cdn.jsdelivr.net/gh/clf99/pong-leaderboard@main/leaderboard.js';

const scores = await Leaderboard.fetchAiScores();
